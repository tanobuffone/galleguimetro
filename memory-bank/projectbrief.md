# galleguimetro

**Dominio**: finance
**Descripción**: Sistema de análisis de opciones financieras en tiempo real

**Objetivo**: Reemplazar el flujo actual de Excel + etrader con una solución más robusta y escalable para análisis de opciones financieras, cálculo de griegas, estrategias de opciones y análisis de portfolio.

**Stack**: Python FastAPI + React + PostgreSQL + Qdrant
**Presupuesto**: 0€ inicial, APIs gratuitas
**Datos**: Conexión DDE desde Excel + APIs gratuitas
