## ¿Por qué existe galleguimetro?

El usuario actualmente utiliza etrader v3.75.9 32 bits en Windows 11 con Excel para análisis de opciones financieras. El flujo actual mediante DDE funciona pero es propenso a romperse con cálculos complejos.

**Problemas a resolver**:
- Fragilidad del sistema Excel con cálculos complejos
- Falta de escalabilidad
- Dificultad para optimizar constantemente
- Necesidad de interfaz web amigable y con buena UX

**Cómo debería funcionar**:
- Mantener conexión DDE actual para datos en tiempo real
- Procesar cálculos complejos en backend robusto
- Mostrar resultados en dashboard web profesional
- Almacenar histórico para análisis y backtesting
