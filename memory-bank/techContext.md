## Stack Tecnológico

**Backend (Python)**:
- FastAPI (framework web)
- QuantLib (cálculo de opciones)
- pandas/numpy (análisis de datos)
- win32com.client (conexión DDE)
- SQLAlchemy (ORM PostgreSQL)
- requests (APIs externas)

**Frontend (React)**:
- React + TypeScript
- Redux Toolkit (estado)
- TradingView Charting Library (gráficos)
- WebSocket (tiempo real)
- Tailwind CSS (estilos)

**Base de Datos**:
- PostgreSQL (ws-postgres:5432)
- Qdrant (ws-qdrant:6333)

**Herramientas**:
- Docker (contenedores)
- Git (control de versiones)
- VS Code (IDE)
