# galleguimetro

Sistema de análisis de opciones financieras en tiempo real

## Descripción

Reemplaza el flujo actual de Excel + etrader con una solución más robusta y escalable para análisis de opciones financieras, cálculo de griegas, estrategias de opciones y análisis de portfolio.

## Stack Tecnológico

**Backend**: Python FastAPI + QuantLib + PostgreSQL
**Frontend**: React + TypeScript + TradingView
**Base de Datos**: PostgreSQL + Qdrant
**Datos**: Conexión DDE desde Excel + APIs gratuitas

## Instalación

1. Clonar el repositorio
2. Instalar dependencias
3. Configurar variables de entorno
4. Iniciar servicios Docker
5. Ejecutar backend y frontend

## Uso

El sistema mantiene la conexión DDE actual desde Excel mientras proporciona una interfaz web profesional para análisis y cálculos complejos.
